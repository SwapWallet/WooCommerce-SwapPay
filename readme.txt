== Description ==
SwapWallet crypto payment gateway for WooCommerce (supports Classic and Blocks checkout).

== Installation ==
1. Upload `woo-swappay` to `/wp-content/plugins/`.
2. Activate **SwapWallet Payment Gateway** in **Plugins**.
3. Configure in **WooCommerce → Settings → Payments → SwapWallet**.
